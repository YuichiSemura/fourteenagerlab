What is this page?

-知識欲を満たすたびに、彼女を思い出す。-

YuichiSemura が Web 系技術勉強を趣味で行う一環として、手始めに自己紹介・ポートフォリオページを作成することにした。 一般に、ポートフォリオサイト的なものを作ろうとすればインターネット上にはそれを簡単に作れるテンプレートがたくさん落ちている。 しかし、それらを使ってどこかで見たことのあるものを作っても面白くない。 本ページのオリジナル要素として、大昔に匿名で書いていたブログで使っていた「中二病的設定」を流用して、ベースとして組み込んでいる。 「ある表裏一体のような2人が所属する、中二病にまつわる理論の構築と実践を行う研究所」という謎の設定を用いて、このFOURTEENAGER Lab.というページは構築されている。 しかし、実態はめちゃくちゃ適当な設定であり、いかにもそれっぽい言葉を並べているだけである。当たり障りがあるからこそ、オリジナルなのである。 ところで、Bootstrapを用いてレスポンシブなデザインを心がけている。2021年11月現在も開発中であり、今後も気が向いた時にコンテンツを更新していく所存である。
