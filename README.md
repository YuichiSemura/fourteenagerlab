## コンセプト

- 厨二っぽい自己紹介ページを作ろう！！

## 環境と技術

- VSCode のでぶこんを使用している。（今の所それほど意味無し）
- Webpack 使ってみたい
- Bootstrap で行けるところまで行こう
- アニメーション、パララックス効果を用いて実装しよう
  - どこかでシングルページな設計もやってみたい

## よく使うコマンド自分用メモ

`npm install`

`npm update`

`npx webpack`

## コンテンツ
